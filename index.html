Department of Dravyaguna
Arka
Botanical Name :

Calotropis gigantea / Calotropis procera

Family Name :

Apocynaceae

Local Name :

Arka

Vernacular Name :

➢Hindi – Aak/Madar
➢English – Giant milkweed/Crown flower

Synonyms :

Arka, Alarka, Mandara, Vasuka

Classical categorization :

➢ Upavisha (semi-poisonous plant)
➢ Kshara dravya & Tikshna dravyas

Habitat :

Widely found in India (dry lands, roadsides, sandy soil)

Morphology :

➢ Large shrub (2–4 m)
➢ Leaves thick, oval
➢ Stem with milky latex
➢ Flowers white/lavender
➢ Fruits green follicles

Useful Parts :

➢ Root bark
➢ Leaves
➢ Flowers
➢ Latex

Chemical Constituents :

➢ Cardiac glycosides
➢ Flavonoids
➢ Triterpenoids
➢ Enzymes

Actions :

➢ Analgesic
➢ Anti-inflammatory
➢ Antimicrobial
➢ Expectorant

Indications :

➢ Skin diseases
➢ Respiratory disorders
➢ Digestive issues
➢ Joint pain

Therapeutic Use :

➢ External application
➢ Internal (under supervision)
➢ Leaves for pain relief

Dose :

➢ Root bark: 125–250 mg
➢ Latex: very small (caution)

Formulations :

➢ Arka Kshara
➢ Arka Taila
➢ Arka Lavana
➢ Arka Ghrita
